Ollama 安装指南

Ollama 是一个轻量级的开源工具，可在您的本地设备上运行各种大型语言模型。本指南将帮助您在不同操作系统上安装和设置 Ollama，并将其与 NativeMind 扩展连接。

在 macOS 上安装 Ollama

步骤 1：下载并安装 Ollama

从 Ollama 官方网站下载 macOS 安装包。

下载 Ollama for macOS

下载完成后，打开 .dmg 文件并将 Ollama 应用程序拖到您的应用程序文件夹中。

步骤 2：启动 Ollama

从应用程序文件夹或 Spotlight 中打开 Ollama。首次启动时，可能会出现安全提示，请允许应用运行。

Ollama 将在后台运行，并在菜单栏中显示图标。

在 Windows 上安装 Ollama

步骤 1：下载并安装 Ollama

从 Ollama 官方网站下载 Windows 安装程序。

下载 Ollama for Windows

下载后运行安装程序，按照安装向导的指示完成安装。

步骤 2：启动 Ollama

安装完成后，Ollama 将自动启动并在系统托盘中运行。

如果没有自动启动，可以从开始菜单中找到并启动 Ollama。

注意： Windows 版 Ollama 需要 Windows 10/11 64 位系统。对于某些模型，可能需要启用 WSL (Windows Subsystem for Linux)。

在 Linux 上安装 Ollama

步骤 1：使用命令行安装 Ollama

打开终端并运行以下命令：

curl -fsSL https://ollama.com/install.sh | sh

步骤 2：启动 Ollama 服务

安装完成后，启动 Ollama 服务：

ollama serve

保持此终端窗口开启，或者您可以将 Ollama 配置为系统服务以在后台运行。

注意： Ollama 在 Linux 上需要 GLIBC 2.35 或更高版本。Ubuntu 22.04+, Debian 12+, Fedora 36+ 等现代发行版都满足此要求。

下载 AI 模型

安装 Ollama 后，您需要下载至少一个 AI 模型才能使用。以下是一些推荐的入门模型：

Qwen3 8B
~4.1GB
ollama pull qwen3:8b
Deepseek R1 1.5B
~1GB
ollama pull deepseek-r1:1.5b
Llama3 8B
~4.7GB
ollama pull llama3:8b

您可以通过终端（macOS/Linux）或命令提示符（Windows）使用上面的命令下载模型，或者直接在 NativeMind 扩展的设置中下载模型。

连接到 NativeMind

安装 Ollama 并下载模型后，按照以下步骤将其与 NativeMind 连接：

步骤 1：检查 Ollama 是否正在运行

确保 Ollama 在您的设备上运行。在 macOS 和 Windows 上，查看系统托盘/菜单栏是否有 Ollama 图标。

步骤 2：在 NativeMind 中配置 Ollama

打开 NativeMind 扩展，点击设置图标，然后：

  1. 在"模型提供者"部分选择"Ollama"
  2. 确认服务器地址为 http://localhost:11434/api（默认地址）
  3. 点击"测试"按钮检查连接
  4. 连接成功后，从下拉列表中选择模型
